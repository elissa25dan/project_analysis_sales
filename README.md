# Анализ продаж интернет-магазина

## Описание проекта
Анализ транзакций онлайн-ретейлера для выявления ключевых метрик, трендов и точек роста бизнеса.

## Данные
- Источник: UCI Online Retail (Kaggle)
- Период: декабрь 2010 — декабрь 2011
- Объем: 541 909 транзакций
- Структура: номер счета, код и название товара, количество, цена, дата, ID клиента, страна

## Стек технологий
- PostgreSQL 15+ — хранение и первичная обработка
- Power Query (Excel) — очистка и подготовка
- Power Pivot (Excel) — модель данных и меры DAX
- Power BI — визуализация и дашборд

## Этапы работы

### 1. SQL (PostgreSQL)
- Создана структура таблицы для хранения данных
- Данные загружены из CSV через команду `\copy`
- Создано представление `clean_orders` с очищенными данными и вычисляемыми полями
- Выполнены аналитические запросы: топ товаров, динамика по месяцам, распределение по странам
- Создано материализованное представление `dashboard_data` для BI-инструментов

### 2. Power Query (Excel)
- Подключение к PostgreSQL и загрузка представления `dashboard_data`
- Проверка и исправление типов данных
- Создание таблицы-календаря

### 3. Power Pivot (Excel)
- Построена модель данных 
- Созданы меры: Total Revenue, Total Orders, Unique Customers, Avg Order Value, Revenue per Customer

### 4. Power BI
- Создан дашборд:
  - Executive Summary — общие KPI и тренды
- Добавлены интерактивные фильтры (слайсеры)
